# Personal Finance Tracker

A Supabase-first React app for tracking income, expenses, budgets, savings goals, and financial trends.

## Stack
- React + Vite + TypeScript
- Tailwind CSS
- React Router
- Supabase
- Recharts

## Getting started
1. Install dependencies.
2. Copy `.env.example` to `.env` and fill in your Supabase values.
3. Run `npm run dev`.

## Supabase
Apply the SQL migration in `supabase/migrations/0001_initial_schema.sql` before connecting the UI to live data.
